UPE IO

UPE IO is a unique opportunity for students to participate in industry related talks, targeted at technologies that aren't typically accessible to students at RPI. These talks benefit them not only as programmers, but as students, and employees as well.

Talk information

The next scheduled talk is on Wednesday, April 24th, at 6:00pm in Lally 102, and will be on the topic of Ruby development.

If you want to get a head start before the talk, please download & install the Ruby language runtime. Instructions for Windows users can be found on rubyinstaller.org. Instructions for OSX users can be found on rvm.io. Linux users, please install Ruby using your package manager of choice.

The preferred version is 1.9, but 2.0 or 1.8 will still work fine for the most part.

Additionally, if you are still having issues setting up Ruby, you may be able to use an online editor, to a certain extent.
